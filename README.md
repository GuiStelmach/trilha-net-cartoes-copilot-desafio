# 💳 Identificador de Bandeira de Cartão de Crédito

Projeto desenvolvido como parte de um desafio da **DIO (Digital Innovation One)** com o objetivo de identificar a **bandeira de um cartão de crédito** a partir do número informado.

A aplicação analisa o padrão numérico do cartão e retorna a bandeira correspondente, como por exemplo **Visa**, **MasterCard**, entre outras.

---

## 📌 Objetivo do Projeto

Este projeto tem como finalidade praticar conceitos de desenvolvimento de software enquanto exploramos o uso de **inteligência artificial no processo de programação**, utilizando o **GitHub Copilot** como assistente de código.

A ideia central do desafio é **reproduzir ou aprimorar um projeto existente**, aplicando boas práticas de programação, versionamento e documentação.

---

## 🚀 Funcionalidades

- Receber um número de cartão de crédito
- Identificar a bandeira com base no padrão do número
- Retornar a bandeira correspondente

### Exemplo de uso

Entrada:

```
4111111111111111
```

Saída:

```
Visa
```
